## Step 1: Get Your API Key (30 seconds)

1. Go to: https://console.anthropic.com/
2. Sign up or log in
3. Click "API Keys" in the left sidebar
4. Click "Create Key"
5. Copy the key (starts with `sk-ant-...`)

## Step 2: Configure Archie (30 seconds)

### Option A: Use the setup script (easiest)
```bash
./setup.sh
```
Paste your API key when prompted.

### Option B: Manual setup
```bash
cp .env.example .env
# Then edit .env and paste your API key
```

## Troubleshooting

### "API Key configured: No"
- Make sure your `.env` file exists
- Make sure it contains `ANTHROPIC_API_KEY=sk-ant-...`
- Restart the server after creating `.env`

### "Failed to generate architecture"
- Check your API key is valid
- Make sure you have API credits in your Anthropic account
- Check the console for error messages

### Port 3000 already in use
Edit `.env` and change:
```
PORT=3001
```
Then access at http://localhost:3001
